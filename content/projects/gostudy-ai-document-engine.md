  GOSTUDY prepares international study applications for students. Every
  application needs a competitive CV and Statement of Purpose built from raw,
  unstructured student material — a process handled entirely by counsellors
  writing documents by hand.
problem: >-
  Producing application documents took 2–4 weeks per student, and quality
  depended on each counsellor's individual writing ability. The manual process
  couldn't scale with student volume and drove production costs up every year.
solution:
  - Designed a three-layer AI-assisted system — material diagnosis, document generation and revision control — converting raw student data into competitive CVs and Statements of Purpose.
  - Built a Google Apps Script pipeline that automatically extracts and structures student data into clean JSON.
  - Paired the pipeline with a custom Claude Skill that counsellors use to generate high-quality first-pass drafts in minutes.
  - Owned the project end-to-end as sole designer and builder — from problem diagnosis through system architecture to implementation.
